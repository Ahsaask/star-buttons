# Team Constellation

An interactive experience where team members are represented as stars. Connect them to discover elemental relationships.

## About

Three team members appear as animated star buttons. Drag between stars to connect them, then unite to reveal constellation with personality descriptions.

### Elemental Bonds
- **Fire**: Ahsaas + Steph
- **Air**: Steph + Michael
- **Water**: Michael + Ahsaas
- **Earth**: All three united

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
